Project: Pizza Order

Pizza Order Page

I coded a pizza-ordering webpage using HTML, JavaScript, and CSS. I used HTML to create the structure of the webpage which includes input fields for the user to put their personal information like their name, address, and card information. There are radio buttons to pick your pizza size, type of crust, and flavor, and there’s a price next to each choice. There’s a submit button to place the order and display a summary of the order. JavaScript was used to process what was put into the pizza order like the input fields and what radio buttons were chosen and display the order summary. Clicking the submit button will calculate the total price depending on what choices were picked, display their selected choices with their corresponding pictures, and display a summary of their order with their personal information. Once the submit button is clicked another button called new order will be displayed and clicking it will reload the page resetting the form. I used CSS to change the font and color of the text that is displayed, adjust the height and width of the input fields, submit button, and new order button. I also used CSS to control the display of the new order button and certain text to be hidden first and then visible when necessary.

Project: Rock Paper Scissors

Rock Paper Scissors Page

I coded a rock-paper-scissors game using HTML, JavaScript, and CSS. I used HTML to create the structure of the webpage displaying rock, paper, and scissor buttons, the CPU choice, and three headings displaying wins, losses, and ties. I used JavaScript to define variables for the counters of the three headings and the CPU choice. I created a CPU function, rock function, paper function, and scissor function. The CPU function randomly selects one of the three options (rock, paper, or scissors). The rock, paper, and scissor functions are used when the user clicks on the buttons. The three button functions each call the CPU function to get the CPU’s choice and display the choice. Then it compares the user’s choice with the CPU’s choice and increases the counter accordingly. The way the game works is simple, rock beats scissors. paper beats rock, and scissors beat paper. So every time a user clicks one of the buttons, the CPU choice will display and the counter headings will be updated based on who won between the CPU and user. I used CSS to make the webpage look better by changing the color of the headers (wins, losses, ties) and setting the images to a certain size.
